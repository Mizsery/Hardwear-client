# Hardwear store react app

### Stack

---

- **RTK**
- **React-router-dom**
- **React-hook-form with zod**
- **Shadcn/UI and tailwind**

### Installation

---

1. Clone repository [HardWear client](https://github.com/Mizsery/Hardwear-client)

```
git clone https://github.com/Mizsery/Hardwear-server
```

2.  Install packages:

```bash
npm i
```
